### Init Repository

```bash
yarn init -y
```

### Install Dependency

```bash
yarn add express mysql2 sequelize cors
```

### Install Monitoring Nodemon

```bash
yarn global add nodemon
```

### Create Database

```sql
CREATE DATABASE crud_db;
```

### Start Backend Service

```bash
nodemon index
```
